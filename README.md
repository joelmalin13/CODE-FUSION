# CodeArena

CodeArena is a web-based coding assessment platform that allows students to solve programming challenges directly in the browser. Code is executed in real-time via Judge0, test cases are evaluated automatically, and results are reflected instantly on a live leaderboard. 

Stuck on a problem? An AI hint system powered by Groq gives contextual nudges without revealing the solution. Built with Next.js, Supabase, Monaco Editor, and Groq API.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
